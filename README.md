# bermudaframework

                                                A little about Bermuda
                                                
  Bermuda is a lightweight minimal style responsive CSS framework. Different themes, text colors and background colors allow for many possibilities.Framework includes a 6 grid responsive layout, button styles, cards and a responsive navbar. Let Bermuda help you create a beautiful responsive website.
  
                                                    Getting Started

1. Download Bermuda. You will get two folders scss and css. Store those folders in your project file.

2. Create your HTML.

3. Link your HTML file to your bermuda.css.

4. Bermuda uses 2 fonts through Google Fonts. 'Playfair Display', serif and 'Poiret One', cursive. Link your HTML to google fonts.

link href="https://fonts.googleapis.com/css2?family=Playfair+Display:ital,wght@0,400;0,700;1,400&family=Poiret+One&display=swap" rel="stylesheet"
