A little about Bermuda

Bermuda is a lightweight minimal style responsive CSS framework. Different themes, text colors and background colors allow for many possibilities.Framework includes a 6 grid responsive layout, button styles, cards and a responsive navbar. Let Bermuda help you create a beautiful responsive website.


Getting Started

1. Download Bermuda. You will get a zip file called bermuda.zip containing two folders scss and css. Store those folders in your project file.

2. Create your HTML.

3. Link your HTML file to your bermuda.css.

4. Bermuda uses 2 fonts through Adobe. Operetta-12 and trade-gothic-next, sans-serif. Link your HTML to Adobe fonts. https://use.typekit.net/myw8bnd.css

Note: You can customize Bermuda in the file called varibles.scss. There you can make changes such as Bermuda colors, text, breakpoints, grid, padding etc. Do not make changes to any other files.

6 Column Grid

BACK TO TOP

Bermuda works on a 6 column responsive grid. You can create a row by creating a div with a class of row. Each column should also have a div with a class of col-1. If you need a way of distinguishing between your columns you can use any class between col-1 - col-6.

Navbar

BACK TO TOP

Setting up your navbar is simple. Just add the class .navbar to your nav. The navbar is responsive. You can make changes to the default nav in the variables.scss file under navbar. Simply replace bermuda values for yours.

Cards

BACK TO TOP

Note: You can make the card responsive just like anything else by adding responsive column classes. Refer to grid for more information on grid.

You can add premade cards with different syles to your website. By default the card has a white background but you have 4 other colors to choose from .card-yellow, .card-red, .card-blue, .card-dark.

Just add a div class of .card. Each element within the card will also require their own class such as .card-header, .card-body and .card-footer

Buttons

BACK TO TOP

Colors

Buttons come in 5 different colors. Give your button a class of the button style you want to use.

Note: Buttons are just one of the things you can customize to your preference in the file variables.scss

Shapes & Sizes

There are two button shapes including the default and the rounded buttons. To get the rounded version simply add rounded to your button class for example .rounded-btn-blue.

There are three size options for buttons. Default which is the largest, .btn-sm and .btn-md. Other features include .over-img-btn, which allows you to position your button over an image.

Note: changes to .over-img-btn positioning as well as other button features can be made in the file variables.scss.

Text Scale & Style

Bermuda is set on 3 different tyscales based on breakpoint. At phone breakpoint the typescale is set to Major Second. At tablet breakpoint the typescale is set to Major third and at desktop it is set to perfect fourth. You can adjust your typescale in the variables.scss file under typescale.

Bermuda also comes with some text styles

.text-italic

.text-allcaps

.text-weight-bold

.text-weight-heavy

.center-text

Others include .over-img-text-center and .over-img-text. These two classes help to center text over images. You can change these values in the variable.scss file.
